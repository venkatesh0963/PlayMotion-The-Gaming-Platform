About Coding Quiz Cyber-Edition

Welcome to the Coding Quiz Cyber-Edition! A true netrunner doesn't just use tools; they understand the raw code that builds the matrix.

How It Works

  • The Core Dump: The system will query you with a question sourced from a vast database of programming languages (JavaScript, HTML, CSS, Python) and computer science fundamentals.
  • Compilation: You will be provided with 4 possible outputs or definitions. Select the correct syntax to compile the node successfully.
  • Real-Time Execution: The terminal instantly evaluates your logic. Valid syntax will compile in neon green. Invalid syntax throws a red exception error while highlighting the correct variable.
  • Session Loop: Each hacking simulation runs a gauntlet of 10 randomized queries to ensure you are continually adapting.
  • Scoring: Achieve a perfect 10/10 to prove your mastery of the mainframe.

Debug your brain, read the documentation carefully, and avoid syntax errors.
