# Play Motion - The Gaming Platform

Welcome to the grid. **Play Motion** is a high-performance, cyberpunk-themed HTML5 arcade platform. It houses a collection of 25 bespoke retro and modern games completely rebuilt with neon aesthetics, fluid mechanics, and dynamic responsive layouts designed for modern browsers.

## 🚀 Features

*   **25 Unique Games**: A massive collection of classic and modern arcade games, including Snake, Flappy Bird, Pong, Tetris, Zombie Survival, Tank Battle, and many more.
*   **Zero Dependencies**: Built entirely with Vanilla HTML5, CSS3, and JavaScript (ES6). No React, no Unity, no complex build steps. Just pure, lightweight code.
*   **Dynamic Theme Toggling**: Seamlessly switch between the default "Dark Neon" mode and "Light Cyber" mode using the `data-theme` API.
*   **HTML5 Canvas Rendering**: All game logic and graphics are drawn dynamically using the ultra-fast 2D canvas context.
*   **Local Storage**: High scores and theme preferences are saved locally on your machine via the `localStorage` API, maintaining a strict zero-knowledge, database-free privacy model.

## 🛠️ How to Play

Running the platform requires absolutely no installation or local server setup.

1.  Clone or download this repository to your local machine.
2.  Open `cyberpunk_gaming.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  Click "Play Now" on any game card to initiate a session.

## 📂 Project Structure

*   `cyberpunk_gaming.html`: The main dashboard and entry point to the arcade.
*   `[game_name]_game.html`: Individual game modules containing their specific canvas logic and UI.
*   `faq.html`, `tos.html`, `privacy.html`: Legal and informational pages mirroring the cyberpunk aesthetic.

## 🎨 Aesthetics & Design

The platform uses a unified design language:
*   **Typography**: Google Fonts (`Orbitron` for headers, `Poppins` for body text).
*   **Color Palette**: High-contrast neon hues (Cyan, Magenta/Pink, Yellow) against deep space-black backgrounds.
*   **Micro-interactions**: Glowing hover effects, smooth CSS transitions, and customized scrollbars.
